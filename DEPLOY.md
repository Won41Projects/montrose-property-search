# Deploy to Render (beginner guide)

See the chat walkthrough for full steps. Quick reference:

- **Start command:** `npm start`
- **Health check:** `/api/version`
- **No build step required** (no npm dependencies to install)

After deploy, open `https://YOUR-APP.onrender.com` and test a search.

To embed on Squarespace, use a Code block with an iframe pointing at that URL.
